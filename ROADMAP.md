## 🔴 Критические пути

### Критические зависимости

```
OAuth настройка → Auth endpoints → Frontend Auth → User Dashboard
PostgreSQL → Backend API → Frontend Integration → Full App
n8n → Workflows → VK Bot → Automation
ЮKassa → Payments → Balance → Monetization
```

### Риски и митигация

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Задержка OAuth аппрува | Средняя | Высокое | Начать регистрацию рано, использовать test mode |
| Проблемы с ЮKassa | Средняя | Высокое | Иметь запасной платёжный шлюз |
| Сложность n8n workflow | Высокая | Среднее | Выделить больше времени на фазу 4 |
| Performance issues | Средняя | Среднее | Раннее тестирование производительности |
| Security уязвимости | Низкая | Высокое | Регулярные security аудиты |
